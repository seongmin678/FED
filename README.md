## 📊 FOMC Minutes Sentiment Analysis (연준 회의록 감성 분석)
This project analyzes Federal Reserve meeting minutes from 2015 to 2026 to quantify "Hawkish" and "Dovish" sentiments and their correlation with interest rate fluctuations.
본 프로젝트는 2015년부터 2026년까지의 연준 회의록을 분석하여 '매파(Hawkish)' 및 '비둘기파(Dovish)' 성향을 정량화하고, 실제 금리 변동과의 상관관계를 입증합니다.

**🚀 Key Features (주요 기능)
NLP Sentiment Analysis: Text mining and sentiment scoring of FOMC minutes using Python.

NLP 감성 분석: Python을 활용한 FOMC 회의록 텍스트 마이닝 및 감성 점수 산출.

Economic Database: Structured storage of historical economic indicators using SQL.

경제 지표 DB: SQL을 이용한 과거 경제 지표 및 회의록 데이터의 구조화된 저장.

Interactive Dashboard: Real-time visualization of sentiment trends and interest rate movements via Streamlit.

인터랙티브 대시보드: Streamlit을 통한 감성 지수 트렌드 및 금리 변동 시각화.

Correlation Research: Proven leading correlation between the "Hawkishness Index" and monetary policy shifts.

상관관계 연구: '매파 지수'와 통화 정책 변화 간의 선행 상관관계 검증.


**🛠 Tech Stack (기술 스택)
Language: Python 3.x

Libraries: NLTK, Scikit-learn, Pandas, NumPy

Database: SQL (PostgreSQL/MySQL)

Visualization: Streamlit, Matplotlib, Plotly


**📈 Key Results (주요 성과)
Quantitative Insights: Successfully quantified qualitative text into a numerical "Hawkishness Index."

정량적 통찰: 질적 텍스트 데이터를 수치화된 '매파 지수'로 변환 성공.

Predictive Value: Identified that sentiment shifts in minutes often precede official interest rate changes by 1-2 quarters.

예측 가치: 회의록의 감성 변화가 실제 금리 변동보다 1~2분기 선행함을 확인.
